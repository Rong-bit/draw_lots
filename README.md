# 專業抽籤系統 Pro Draw

一個功能豐富、介面美觀的線上抽籤系統，支援多種抽獎模式和自訂設定。

## 功能特色

- 🎯 **多種抽獎方式**：分次抽出、一次抽完、倒序抽獎
- 🎲 **靈活的中獎規則**：防重複中獎、權重機率、名單管理
- 🎊 **豐富的視覺效果**：彩帶動畫、音效、抽獎動畫
- 📊 **完整的結果管理**：即時顯示、匯出 CSV、歷史記錄
- 📱 **響應式設計**：支援桌面和行動裝置
- ⚙️ **規則設定 Modal**：集中管理所有設定選項

## 快速開始

### 本地開發

**需求：** Node.js 18 或更高版本

1. 安裝依賴：
   ```bash
   npm install
   ```

2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

3. 在瀏覽器中開啟 `http://localhost:3000`

### 建置生產版本

```bash
npm run build
```

建置檔案將輸出到 `dist` 目錄。

### 預覽生產版本

```bash
npm run preview
```

## 使用說明

1. **設定主題**：在頂部輸入框輸入抽獎活動主題
2. **輸入獎項**：在「獎項內容」區塊輸入獎項名稱和數量（格式：`獎項名稱 數量`）
3. **輸入參與者**：在「參加名單」區塊輸入參與者名單（每行一位）
4. **調整規則**：點擊頂部設定按鈕開啟規則設定 modal
5. **開始抽獎**：點擊「開始抽獎」按鈕進行抽獎
6. **匯出結果**：抽獎完成後可匯出 CSV 檔案

## 技術棧

- **React 19** - UI 框架
- **TypeScript** - 類型安全
- **Vite** - 建置工具
- **Tailwind CSS** - 樣式框架
- **Lucide React** - 圖示庫
- **Canvas Confetti** - 彩帶動畫

## 部署

此專案已配置 GitHub Actions，自動部署到 GitHub Pages。

1. 在 GitHub 上建立新的倉庫
2. 推送代碼到 `main` 分支
3. 在倉庫設定中啟用 GitHub Pages
4. 選擇 `gh-pages` 分支或 GitHub Actions 作為來源
